# Air Quality Changelog

## [Initial Version] - {PR_MERGE_DATE}

- Add Check Air Quality command to view detailed air quality data for any city
- Add Top Polluted Cities command to view the top 25 most polluted cities
- Support for default city preference
- Display AQI, level, main pollutant, and individual pollutants (PM2.5, PM10, O₃, NO₂, SO₂, CO)
- Color-coded AQI levels with visual indicators
- Direct navigation from top cities list to city details
- HTML scraping implementation using cheerio for reliable data access
